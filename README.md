# GameRix - Gaming Dashboard (v1.0)
by Shub Shinde


## Installation Steps
1. Clone this repository.
2. Run `npm install` to download all the required dependencies.
3. Run `npm run start` or `yarn start` to run the project.


## Technologies Used
1. React.js
2. SCSS
3. Splide Sliders


## Available Scripts
In the project directory, you can run:
`npm start`
`npm run build`
`npm run eject`
